# 📚 Projeto de Redes Neurais

## 📖 Descrição
Este repositório é dedicado ao estudo da disciplina de **Redes Neurais**, cursada durante a graduação em Ciência da Computação.

O objetivo principal é centralizar implementações, experimentos e anotações relacionadas a conceitos fundamentais e avançados de redes neurais, servindo como apoio prático ao aprendizado teórico.

---

## 🎯 Objetivos
- Consolidar conceitos de **Machine Learning** e **Deep Learning**
- Implementar modelos de redes neurais do zero e com frameworks
- Explorar diferentes arquiteturas e técnicas de treinamento
- Documentar aprendizados e insights obtidos ao longo da disciplina

---

## 🧠 Conteúdos abordados
- Redes neurais artificiais básicas
- Funções de ativação
- Algoritmo de backpropagation
- Treinamento e validação de modelos
- Overfitting e regularização
- Redes neurais profundas (Deep Learning)
- Introdução a frameworks (ex: TensorFlow, PyTorch)

---

## 🛠️ Tecnologias utilizadas
- Python
- NumPy
- Pandas
- TensorFlow / PyTorch
- Jupyter Notebook
